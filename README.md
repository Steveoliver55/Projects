The firewall, IPS, VPN's and endpoints (FIVE) project was designed to help us understand,
analyze and respond to various networking alerts and log levels in a simulated network
attack. 

The Security Mechanisms project is an exercise into how to secure a building and all assets contained within.

The Vulnerabilities and exploits project involved successfully gaining root access on a linux target machine using various hacker tools such as metasploit.

The Wireless and Mobile Device Project involved hacking into three routers of varying difficulty and gaining a username and passsword to successfully connect to the routers over Wi-Fi.
